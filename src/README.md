# 🛒 SISTEMA DE GESTIÓN Y ATENCIÓN PRESENCIAL DE EVENTOS - RETAIL
**Proyecto Ingeniería de Software | Arquitectura de Microservicios**

## 📋 DESCRIPCIÓN
Sistema para la gestión completa de turnos, pronóstico inteligente de tiempos de espera, asignación de recursos y notificaciones para establecimientos comerciales.

## 🛠️ TECNOLOGÍAS UTILIZADAS (TODAS OPEN SOURCE)
- ✅ **Backend**: Python 3.11 + FastAPI
- ✅ **Base de Datos**: PostgreSQL
- ✅ **Arquitectura**: Microservicios
- ✅ **Control de Versiones**: GitHub
